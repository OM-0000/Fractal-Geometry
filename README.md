# Fractal-Geometry

# Overview
Interactive fractal and mathematical visualization tool built in C++ with raw Win32/GDI and a custom UI toolkit. Includes L‑system trees, Barnsley Fern, and golden‑ratio phyllotaxis with adjustable parameters, text essays, and image slides.

# Fractal & Mathematical Visualization Showcase  
This project is an interactive visualization system built entirely in C++ using raw Win32/GDI and a custom UI toolkit I developed from scratch (buttons, sliders, panels, event routing, and internal movable windows). It contains three independent mathematical modules—L‑system fractal trees, the Barnsley Fern, and golden‑ratio phyllotaxis—each with adjustable parameters, real‑time rendering, and accompanying educational content.

The application includes a modular interface with a button bar for switching between simulations, a property‑sheet system for controlling parameters, and built‑in text and image viewers for reading essays and exploring reference images. All rendering, UI behavior, and memory management are implemented manually without external frameworks.

Originally created for an Intro to Biology class to demonstrate fractal patterns in nature, the project has since evolved into a polished, self‑contained mathematical visualization tool.
